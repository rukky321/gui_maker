# GUI Maker

Minecraft Datapack for JE v1.21.5

## 何ができるの？

エンダーチェストを使ってGUIを作成できるようになるデータパックです。

## 導入方法

一般的なマインクラフトデータパックと同様にdatapacksフォルダ内にこのデータパックを入れれば導入完了です。

## 使い方

[wiki](https://github.com/rukky321/gui_maker/wiki)見てね。
